AppCode 管联 CocoaPods 的Bug

发表于 2017-01-18 | |

AppCode 中添加了直接对CocoaPods的关联：

但是直接关联 Ruby SDK 和 CocoaPods 的时候会发现提示错误：

照着错误提示修改 AppCode 里面 cocoapods_common.rb 和 cocoapods_list,rb 里的Pod::SourcesManager 都切换为 Pod::Config.instance.sources_manager 就可以了。

AppCode 里面的 Bug 其实还真的不少，自动填充代码的功能一直没修好，这个应该算是CocoaPods的锅，开源的东西自己没事瞎改。
